# Smart Queue Management System

A Firebase-based Android Queue Management App built using Java and Android Studio.

## Features
- User Authentication
- Generate Queue Tokens
- Real-time Queue Updates
- Admin Dashboard
- Next Token Management
- Complete Token Handling
- Firebase Realtime Database Integration

## Tech Stack
- Java
- Android Studio
- Firebase Authentication
- Firebase Realtime Database
- RecyclerView
- Material UI

## Setup Instructions
1. Clone the repository

4. Sync Gradle
5. Run the app
